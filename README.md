### 自用的模拟 vue-cli 脚手架
### 常用命令
 - notel add templateTypeName projectUrl -- 添加模板
 - notel delete -- 删除模板
 - notel list -- 查看所有模板
 - notel init templateTypeName projectName -- 初始化项目
### npm 发包命令
 - npm version patch 1.0.0 -> 1.0.1 --小版本升级补丁
 - npm version minor 1.0.0 -> 1.1.0 --次要补丁
 - npm version major 1.0.0 -> 2.0.0 --主要版本
 - npm publish 发布更新
